RobloxPI

API's

  • https://robloxpi.onrender.com/v1/users?id=useridhere -- User Info
  • https://robloxpi.onrender.com/v1/users/username-history?id=useridhere -- User's Username History
  • https://robloxpi.onrender.com/v1/games?universeId=UNIVERSEIDHERE -- Game Info
  • https://robloxpi.onrender.com/v1/games/thumbnails?universeId=UNIVERSEIDHERE -- 512x512 Icon Thumbnail
  • https://robloxpi.onrender.com/v1/games/votes?universeId=UNIVERSEIDHERE -- Game Votes
RobloxPI is a API for Roblox. This third-party API is not a offical Roblox API. But this does use Roblox APIs.
